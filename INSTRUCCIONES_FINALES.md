## 🚀 **Cómo Usar el Sistema:**

### 1. **Descarga y Configuración (5 minutos)**
```bash
# Descarga el ZIP y extrae los archivos
# Hacer ejecutable el script de configuración
chmod +x configurar-entorno.sh

# Ejecutar configuración automatizada
./configurar-entorno.sh

# Configurar tus credenciales
cp .env.ejemplo .env
# Edita .env con tus valores reales
```

### 2. **Configurar Google Services (10 minutos)**
```bash
# Configurar Google Sheets y Drive
npm run configurar-todo

# Actualizar configuración N8N
node actualizar-config-n8n.js
```

### 3. **Subir Automáticamente a GitHub (2 minutos)**
```bash
# Ejecutar script de GitHub (te guía paso a paso)
chmod +x configurar-github.sh
./configurar-github.sh
```

### 4. **Importar a N8N (5 minutos)**
- Copia el contenido de `sistema-ia-legal-n8n-workflow.json`
- Importa en tu instancia de N8N
- Configura las credenciales
